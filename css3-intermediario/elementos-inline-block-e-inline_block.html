Os comportamentos padrão são:
1) Block => cobre toda width, sequenciados formando uma coluna (isto é, cada novo elemento é uma nova linha)

2) Inline => cobre apenas o tamanho do conteúdo que encerra, sequenciados formando uma linha (isto é, cada novo elemento é uma nova coluna)

3) Inline-Block => cobre apenas o tamanho do conteúdo que encerra, sequenciados formando uma coluna (isto é, cada novo elemento é uma nova linha), apenas quando o tamanho da width tenha sido atingida (devido ao preenchimento do block)
